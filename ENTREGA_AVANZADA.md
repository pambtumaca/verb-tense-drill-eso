## Descripción breve para Aules

Se entrega la version final del recurso "Verb Tense Interactive Drill" publicada en GitHub Pages. La actividad interactiva esta completamente en ingles y permite completar frases, corregir cada respuesta con feedback inmediato y mostrar una puntuacion final. El README incluye objetivo didactico, nivel destinatario e instrucciones de uso en clase. El recurso esta preparado para su uso directo en aula en una sesion de repaso de present simple y past simple.

## Comprobaciones realizadas

1. Flujo funcional probado en local (respuestas correctas e incorrectas).
2. Mensajes de feedback claros y puntuacion final operativa.
3. README actualizado en castellano para uso docente.
4. Cambios subidos a la rama main en GitHub.
