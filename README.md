# Verb Tense Interactive Drill

## Objetivo didáctico

Este recurso permite practicar en inglés la forma verbal correcta en present simple y past simple mediante frases breves y corrección inmediata. Está pensado para repasar de forma rápida y visual, con una dinámica sencilla que facilita el trabajo autónomo o guiado en el aula.

## Nivel o grupo destinatario

Alumnado de ESO, especialmente 1.º y 2.º de ESO, o cualquier grupo que necesite un repaso inicial de present simple y past simple.

## Instrucciones de uso en clase

1. Proyecta la actividad al inicio de la sesión o como repaso final.
2. Pide al alumnado que complete cada hueco con la forma verbal correcta.
3. Haz clic en Check para mostrar el feedback inmediato en pantalla.
4. Usa el mensaje de error para corregir en voz alta y reforzar la forma correcta.
5. Termina con Show final score como comprobación rápida o autoevaluación.

## Cómo probarlo en local

1. Abre la carpeta del proyecto en VS Code.
2. Abre index.html en el navegador o con Live Server.
3. Prueba respuestas correctas e incorrectas para comprobar el feedback.
4. Pulsa Show final score para revisar la puntuación global.

## Texto breve para la entrega

Este recurso se puede usar como actividad de inicio, repaso o cierre en una sesión de inglés en ESO. El alumnado completa formas verbales en presente y pasado simple y recibe feedback inmediato en pantalla. Esto permite corregir errores al momento y comentar la respuesta correcta de forma guiada o autónoma. La actividad también puede emplearse como autoevaluación rápida al final de la clase.
